Module 06: 品牌声誉风险大屏与舆论传播机制可视化 (Reputation Dashboard & Viral Mechanics)
🎯 模块业务目标：
本模块是全链路分析的“视觉结案”。我们将 05 模块产出的统计学指标转化为具有高度商业直觉的交互式看板（BI Dashboard）。通过宏观分布、表达欲透视、风险成分拆解、传播动力学四个维度，全方位复现 YouTube 舆论场的真实面貌。

🚨 指标严谨性对齐声明：

纠正置信度幻觉：将 04 模块原始输出的 sentiment_score 严格区分为：

【真实情感极性 (True Sentiment)】：映射至 -1.0 (极负) 到 1.0 (极正)，用于衡量情绪方向。

【情绪纯度 (Emotion Purity)】：0.5 到 1.0，用于衡量情绪的明确性/坚定程度。

【负面共识度 (Negative Consensus)】：仅针对负面评论池，衡量指控的统一性。

指标口径对齐：所有图表文案均与 05 模块的【品牌声誉风险计分卡】保持 100% 一致。

📌 Cell 1 - BI 视觉引擎初始化与全量数据装载 (Initialization)
职责：

挂载 plotly 高级交互绘图引擎，定义符合品牌声誉预警的全局配色方案（红色-风险、绿色-安全、蓝色-中立）。

跨模块数据锚定：同步装载 04 模块语料底座与 05 模块计算出的最新【风险计分卡资产】及【全局相关性矩阵】。

📊 Cell 2 - 宏观舆情大盘：100% 情感堆叠分布图 (Macro Sentiment Landscape)
职责：展示三大控制组（SHEIN、对照品牌、环保基准线）的情感比例构成。

业务指标：

情感分布比例 (Sentiment Ratio)：对比各品牌负面情绪的“基本盘”厚度。

商业叙事：直观展示 SHEIN 与对照组在“负面率”上的量级差异，确立品牌声誉受损的基础物理空间。

📊 Cell 3 - 表达欲透视：不同情感下的平均评论长度 (User Effort Analysis)
职责：对比不同情感标签下的 【平均评论长度 (Average Text Length)】。

指标对齐：使用 05 模块定义的 User Effort。

商业叙事：视觉化验证“愤怒是否驱动了更高的表达成本”。若负面评论字符数显著高于正/中性评论，则证明该品牌正面临“高卷入度（High Involvement）”的情绪宣泄。

🚨 Cell 4 - 风险成分拆解：品牌声誉风险计分卡瀑布流 (Risk Scorecard Decomposition)
职责：对 05 模块产出的 REPUTATION RISK SCORE (0-100) 进行各维度贡献度拆解。

四个核心子维度（严谨对齐 05 权重）：

NSS 风险 (30%)：口碑护城河受损深度。

点赞加权负面率 (50%)：流量引爆的致命杀伤力（核心引擎）。

绝对负面率 (15%)：舆论波及的广域曝光度。

负面共识度 (5%)：网民指控的一致性与语义纯粹性。

视觉形式：采用堆叠条形图（Stacked Bar），将各维度折算出的“风险分值”进行累加，直观对比三组品牌的“致命伤”分布。

🌪️ Cell 5 - 病毒传播机制：极性、纯度与流量杠杆 (Viral Mechanics Visualization)
职责：

A. 全局相关性热力图：呈现 【真实情感极性 (True Sentiment)】、【情绪纯度 (Emotion Purity)】 与 【点赞量 (Likes)】 的统计学联系。

B. 流量敏感度曲线 (LOWESS)：展示在不同情感方向下，评论长度（表达欲）对点赞量（引爆力）的“边际共鸣收益”。

指标纠偏逻辑：

X 轴：彻底抛弃 0.5-1.0 的原始置信度刻度，重构为 -1.0 到 1.0 的极性轴。

Y 轴：采用对数刻度（Log Scale）展示点赞数，以适配长尾分布。

商业叙事：视觉化实锤“负面且情绪明确的中长文”是引爆 YouTube 推荐算法、造成品牌二次伤害的“病毒传染源”。
